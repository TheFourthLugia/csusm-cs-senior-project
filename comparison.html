Below shows a diagram of each subject of computing as well as computer-related majors (including Software Engineering and Computer Engineering) and where they fall into a broad area of computing. Computer Information Systems primarily deals with transformation of data using pre-built software, while Computer Science primarily deals with the foundations of software and cybersecurity.

Potential Jobs

A student graduating with a CS degree may best suit the following jobs:

  • Business Systems Analyst
  • DevOps Engineer
  • Embedded Software Engineer
  • Full Stack Developer
  • Game Developer
  • Java Developer
  • Principal Software Engineer
  • Principal Systems Engineer
  • Software Developer
  • Software Engineer
  • Systems Engineer

A student graduating with a CIS degree may best suit the following jobs:

  • Data and Analytics Engineer
  • Database Administrator
  • Database Designer/Developer
  • Information Systems Security Engineer
  • Information Systems Developer
  • Information Systems Analyst
  • Network Administrator
  • Network Designer
  • Web Developer
